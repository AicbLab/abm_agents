# ABM 消费决策依赖梯度仿真系统 - 前端交互平台

## 📋 功能特性

### ✨ 核心功能
- **🎛️ 参数调节**：通过滑动条实时调整 5 种依赖等级的初始比例（L1-L5）
- **▶️ 仿真控制**：开始、暂停、重置、停止，完全掌控仿真流程
- **🐢🐇 速度调节**：5 档速度可选（0.5x, 1x, 2x, 5x, 10x），手动控制播放速度
- **📊 实时可视化**：动态展示依赖等级分布和系统指标演化
- **📈 数据统计**：实时显示步数、满意度、AI 使用率等关键指标

### 🎨 界面设计
- **现代化 UI**：渐变色彩、圆角设计、阴影效果
- **响应式布局**：自适应不同屏幕尺寸
- **交互式控件**：悬停动画、平滑过渡
- **数据可视化**：柱状图、折线图、进度条

## 🚀 快速启动

### 方法一：直接打开（推荐）
直接双击 `index.html` 文件即可在浏览器中打开使用。

### 方法二：使用 Python 服务器
```bash
# 进入 frontend 目录
cd frontend

# 启动服务器
python start_server.py
```

服务器会自动打开浏览器并访问：http://localhost:8000

## 🎮 使用说明

### 1️⃣ 设置初始参数
- 拖动 5 个滑动条调整各依赖等级的比例
- L1（自主型）、L2（信息辅助）、L3（半委托）、L4（高度依赖）、L5（完全代理）
- 确保总比例为 100%（会有提示）

### 2️⃣ 开始仿真
- 点击"▶️ 开始仿真"按钮
- 观察依赖等级分布的实时变化
- 查看系统指标演化曲线

### 3️⃣ 控制播放
- **暂停/继续**：点击"⏸️ 暂停"按钮
- **速度调节**：选择 0.5x / 1x / 2x / 5x / 10x
- **重置**：点击"🔄 重置"回到初始状态
- **停止**：点击"⏹️ 停止"结束仿真

### 4️⃣ 观察指标
- **当前步数**：仿真进行到的步数
- **智能体总数**：系统中的 Agent 数量（默认 500）
- **平均满意度**：所有智能体的平均满意度（0-1）
- **AI 使用率**：使用 AI 代理的智能体比例

## 📊 可视化说明

### 依赖等级分布图
- 5 个柱子代表 L1-L5 五个等级
- 高度表示该等级的智能体数量
- 顶部数字显示具体数量
- 颜色采用渐变紫色系

### 系统指标演化图
- **蓝色线**：满意度趋势
- **紫色线**：AI 使用率趋势
- **红色线**：错误率趋势
- X 轴：仿真步数
- Y 轴：指标值（0-1）

## ⚙️ 配置说明

### 仿真参数
- **智能体总数**：500（可在代码中修改）
- **最大步数**：300 步（可在界面上修改）
- **默认速度**：1x

### 速度档位
- **0.5x**：慢速播放，适合仔细观察
- **1x**：正常速度
- **2x**：加速播放
- **5x**：快速播放
- **10x**：极速播放，快速看到结果

## 💡 使用技巧

1. **参数调优**：先设置为均匀分布（各 20%），然后逐步调整观察效果
2. **对比实验**：多次重置，尝试不同的参数组合
3. **细节观察**：使用 0.5x 慢速观察每一步的变化
4. **快速预览**：使用 10x 速度快速查看整体趋势
5. **暂停分析**：在关键节点暂停，仔细查看数据

## 📝 注意事项

- ✅ 确保 5 个等级的比例总和为 100%
- ✅ 建议使用 Chrome、Edge、Firefox 等现代浏览器
- ✅ 首次加载可能需要几秒钟
- ✅ 如果图表不显示，请检查浏览器控制台是否有错误

## 🔧 故障排除

### 问题：页面空白
**解决**：检查浏览器控制台，确保 JavaScript 已启用

### 问题：图表不显示
**解决**：检查网络连接（需要从 CDN 加载 Chart.js 库）

### 问题：服务器无法启动
**解决**：检查端口 8000 是否被占用，可修改 `start_server.py` 中的 PORT 变量
